# SQL Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow using SQL. The analysis begins with Exploratory Data Analysis (EDA) to understand the structure, quality, and characteristics of the dataset, followed by Advanced Analytics techniques to uncover trends, patterns, and actionable business insights.

The repository contains a collection of SQL scripts organized into analytical modules, showcasing practical SQL skills commonly used in Data Analytics, Business Intelligence, and Data-Driven Decision Making.

---

## Project Workflow

### Exploratory Data Analysis (EDA)

The first phase focuses on understanding the dataset and identifying key business metrics.

#### 1. Database Exploration

* Explore database structure
* Identify tables and relationships
* Understand data availability

#### 2. Dimensions Exploration

* Analyze categorical attributes
* Examine business dimensions
* Discover unique values

#### 3. Date Exploration

* Investigate time ranges
* Analyze historical coverage
* Validate temporal consistency

#### 4. Measures Exploration

* Explore numerical metrics
* Calculate key business measures
* Identify high-level KPIs

#### 5. Magnitude Analysis

* Compare values across categories
* Measure distribution and scale
* Detect significant contributors

#### 6. Ranking Analysis

* Top-N and Bottom-N analysis
* Best and worst performers
* Comparative rankings

---

### Advanced Analytics

The second phase transforms raw data into meaningful business insights.

#### 7. Change Over Time Analysis

* Trend identification
* Growth and decline patterns
* Time-series analysis

#### 8. Cumulative Analysis

* Running totals
* Rolling metrics
* Progressive performance tracking

#### 9. Performance Analysis

* KPI evaluation
* Benchmark comparisons
* Performance monitoring

#### 10. Part-to-Whole Analysis

* Percentage contribution analysis
* Market share calculations
* Category contribution assessment

#### 11. Data Segmentation

* Customer segmentation
* Product segmentation
* Group-based analysis

#### 12. Reporting & Insights

* Business reporting
* Summary dashboards
* Actionable recommendations
